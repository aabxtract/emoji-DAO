## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke trust and sophistication.
- Background color: Dark gray (#222222) for a clean, modern dark theme.
- Accent color: Vibrant magenta (#FF00FF) to highlight interactive elements and voting options.
- Body and headline font: 'Inter', a grotesque-style sans-serif, to offer a modern, objective feel.
- Use emojis consistently to represent voting options and results.
- Proposals displayed as individual cards with clear vote options and real-time counts.
- Subtle animation to indicate successful vote submission.
